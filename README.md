## Server

As a server we use Firebase. Server repo you can find here: https://dziobakwszafie.github.io/ski-service-server/

### To run it locally

Get the repo

```
git clone https://dziobakwszafie.github.io/ski-service/
```

or

```
download ZIP file
```

Then run npm scripts at root folder

```
npm install
npm start
```
