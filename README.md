## Getting Started

Live version you can find here: https://dziobakwszafie.github.io/ski-service/

### To run it locally

Get the repo

```
git clone https://dziobakwszafie.github.io/ski-service/
```

or

```
download ZIP file
```

Run npm scripts

```
npm install
npm start
```
